No Filter Coffee Android Studio starter project
Stack: Java + XML + RecyclerView + Material/AppCompat.

How to use:
1. Open Android Studio.
2. Choose Open.
3. Select this folder: NoFilterCoffee_AndroidStudio_JavaXML
4. Let Gradle sync.
5. Run app.

Important:
- Current images are placeholder drawables.
- Replace @drawable/no_filter_placeholder with cropped product images from No Filter Instagram.
- Firebase is not added yet. This is the front-end working starter with dummy data.
- Package name: com.example.nofiltercoffee
